## Solution

```java
class Solution {
    public int repeatedStringMatch(String a, String b) {
        int aLen = a.length();
        int bLen = b.length();

        int count = 0;
        String res = "";

        while(res.length() < bLen){
            res += a;
            count++;
        }
        
        if(res.contains(b)) return count;

        res+=a;
        if(res.contains(b)) return count+1;
        return -1;
    }
}
```
