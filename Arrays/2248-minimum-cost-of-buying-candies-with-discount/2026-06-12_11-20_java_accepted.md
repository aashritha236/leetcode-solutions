## Solution

```java
class Solution {
    public int minimumCost(int[] cost) {
        
        int len = cost.length;
        if(len <= 2)    return Arrays.stream(cost).sum();

        Arrays.sort(cost); //sorts the array in ascending

        int total = 0;
        for(int i = len-1; i >= 0; --i){
            if((len-1-i)%3 != 2)
                total += cost[i];
        }
        return total;
    }
}
```
