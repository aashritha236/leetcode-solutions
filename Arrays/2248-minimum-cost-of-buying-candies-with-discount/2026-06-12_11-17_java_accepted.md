## Solution

```java
class Solution {
    public int minimumCost(int[] cost) {
        
        int len = cost.length;
        if(len <= 2)    return Arrays.stream(cost).sum();

        Arrays.sort(cost); //sorts the array in ascending
        int left = 0, right = cost.length - 1;

        //Reversing the order of the array in-place to make it descending.
        while(left < right){
            int temp = cost[left];
            cost[left] = cost[right];
            cost[right] = temp;
            left++;
            right--;
        }      

        int total = 0;
        for(int i = 0; i < len; i++){
            if(i%3!=2)
                total += cost[i];
        }
        return total;
    }
}
```
