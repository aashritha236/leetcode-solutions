## Solution

```java
class Solution {
    public boolean isAnagram(String s, String t) {
        int len1 = s.length();
        int len2 = t.length();

        char[] sArray = s.toCharArray();
        char[] tArray = t.toCharArray();

        Arrays.sort(sArray);
        Arrays.sort(tArray);

        if(len1!=len2)
            return false;

        for(int i = 0; i < len1; i++){
            if(sArray[i] != tArray[i])
                return false;
        }
        return true;
    }
}
```
