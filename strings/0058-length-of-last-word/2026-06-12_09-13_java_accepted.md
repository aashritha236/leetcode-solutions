## Solution

```java
class Solution {
    public int lengthOfLastWord(String s) {
        if(s.length() == 0) return 0;

        int i = s.length() - 1;
        int wordlength = 0;

        while(i >= 0 && s.charAt(i) == ' '){
            i--;
        }
        
        while(i>=0 && s.charAt(i)!=' '){
            wordlength++;
            i--;
        }
        return wordlength;
    }
}
```
