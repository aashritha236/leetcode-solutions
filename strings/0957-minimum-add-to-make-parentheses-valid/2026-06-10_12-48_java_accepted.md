## Solution

```java
class Solution {
    public int minAddToMakeValid(String s) {
       
        Stack <Character> stack = new Stack<>();
        int i = 0;

        for(char ch:s.toCharArray()){
            if(ch == '(')
                stack.push(ch);

            else if(ch == ')'){
                if(stack.isEmpty()) i++;
                else    stack.pop();
            }
        }

        return i + stack.size();
    }
}
```
